# AI Summary and Insights

## AI Tools Used
We used ChatGPT as an AI coding and planning assistant during Exercise 9.

## How AI Was Used
- Transforming the MAYIMTECH project idea into a focused live demo.
- Writing English requirements before coding.
- Generating HTML, CSS, and JavaScript code for two web applications.
- Planning the Firebase Realtime Database structure.
- Improving UI clarity for an in-class demo.
- Creating demo flow, user stories, and presentation wording.

## Human Decisions
The team decided the product context, the patient/caregiver scenario, the demo flow, and the final scope. We chose to keep the MVP simple: one patient, one caregiver, simulated readings, and threshold-based alert logic.

## Key Insights
1. Good requirements made the AI output more accurate and reduced rework.
2. A simple MVP is better for a live class demo than a complex system with too many features.
3. Firebase Realtime Database is useful for demonstrating live synchronization quickly.
4. Separating the project into HTML, CSS, and JavaScript files made the code easier to understand.
5. The demo should focus on the two required user stories rather than unnecessary features.
