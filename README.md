# MAYIMTECH Demo

Two-app live demo for Exercise 9:
- `patient.html` - smartphone patient app.
- `caregiver.html` - laptop caregiver dashboard.

## How to Run
Open `patient.html` and `caregiver.html` in the browser.

## Demo Flow
1. Open patient app on phone.
2. Open caregiver dashboard on laptop.
3. Click `Force Alert` or `Simulate Dehydration` in patient app.
4. Caregiver dashboard updates within a few seconds.

## Firebase
The project uses Firebase v8 compat CDN libraries and Firebase Realtime Database.
